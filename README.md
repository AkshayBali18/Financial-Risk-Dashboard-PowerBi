# 📊 Financial Risk Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing financial performance and credit risk using Power BI.

## 🎯 Objectives
- Analyze company financial health
- Identify risk using financial ratios
- Visualize KPIs like:
  - Net Profit Margin
  - Debt-to-Equity Ratio
  - Return on Equity (ROE)
  - Interest Coverage Ratio

## 📂 Dataset
Includes:
- Financial Report

## 📊 Dashboard Features
- KPI Cards
- Trend Analysis (YoY)
- Risk Segmentation
- Dynamic Filters & Drill-through

## 🛠 Tools Used
- Power BI
- DAX
- Power Query

## 📈 Key Insights
- Companies with high debt have lower interest coverage
- Strong correlation between ROE and credit rating

## 🚀 How to Use
1. Download `.pbix` file from `/pbix`
2. Open in Power BI Desktop
3. Refresh data if needed
